下面列一些“大文档富文本编辑器”常见操作，按类型整理，包含常见快捷键参考。快捷键可按平台适配：Windows/Linux 通常是 `Ctrl`，macOS 通常是 `Cmd`。

## 1. 光标移动

| 操作 | Windows/Linux | macOS |
|---|---|---|
| 左/右移动一个字符 | `←` / `→` | `←` / `→` |
| 上/下移动一行 | `↑` / `↓` | `↑` / `↓` |
| 移动到行首/行尾 | `Home` / `End` | `Cmd + ←` / `Cmd + →` |
| 按单词移动 | `Ctrl + ←/→` | `Option + ←/→` |
| 移动到文档开头/结尾 | `Ctrl + Home/End` | `Cmd + ↑/↓` |
| 移动到段落前后 | 常见为 `Ctrl + ↑/↓` | `Option + ↑/↓` |

## 2. 文本选择

| 操作 | Windows/Linux | macOS |
|---|---|---|
| 按字符扩展选择 | `Shift + ←/→` | `Shift + ←/→` |
| 按行扩展选择 | `Shift + ↑/↓` | `Shift + ↑/↓` |
| 选中到行首/行尾 | `Shift + Home/End` | `Shift + Cmd + ←/→` |
| 按单词扩展选择 | `Ctrl + Shift + ←/→` | `Option + Shift + ←/→` |
| 选中到文档开头/结尾 | `Ctrl + Shift + Home/End` | `Cmd + Shift + ↑/↓` |
| 全选 | `Ctrl + A` | `Cmd + A` |
| 鼠标拖拽选择 | 拖拽 | 拖拽 |
| 双击选中词 | 双击 | 双击 |
| 三击选中段落 | 三击 | 三击 |

| 操作 | Windows/Linux | macOS |
|---|---|---|
